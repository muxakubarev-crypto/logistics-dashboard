# Как бесплатно разместить дашборд в интернете (Vercel)

Vercel — это бесплатный хостинг для сайтов. Ты загружаешь код → сайт появляется в интернете по ссылке. Всё просто и бесплатно.

---

## Шаг 1: Установи программу Git (один раз)

Git — программа, которая отправляет твой код на сервер Vercel.

1. Скачай Git отсюда: https://git-scm.com/download/win
2. Нажми «64-bit Git for Windows Setup» — начнётся загрузка
3. Запусти скачанный файл и жми «Next» → «Next» → «Next» (все настройки по умолчанию подходят)
4. В конце нажми «Install»

---

## Шаг 2: Зарегистрируйся на GitHub (один раз)

GitHub — сайт, где хранится твой код. Бесплатно.

1. Зайди на https://github.com
2. Нажми «Sign up» (зелёная кнопка)
3. Введи свой email, придумай пароль и имя пользователя (например: `MuxaKub`)
4. Нажми «Create account»
5. Подтверди email (придёт письмо — нажми на ссылку в нём)

---

## Шаг 3: Установи Vercel CLI (один раз)

Открой командную строку:
- Нажми `Win + R`
- Введи `cmd`
- Нажми Enter

В чёрном окне введи команду и нажми Enter:
```
npm install -g vercel
```
Подожди 1-2 минуты, пока установится.

---

## Шаг 4: Первый деплой (загрузка сайта)

В том же чёрном окне (или открой новое `Win+R → cmd`) перейди в папку проекта:

```
cd C:\Users\Mike\Desktop\WorkDASHBOARD
```

Теперь выполни по очереди 5 команд (копируй и вставляй, нажимай Enter после каждой):

```
git init
```
*(эта команда создаёт пустой Git-репозиторий в папке)*

```
git add .
```
*(добавляет все файлы проекта для отправки)*

```
git commit -m "Первая версия дашборда"
```
*(сохраняет текущее состояние файлов)*

Теперь запусти деплой:
```
vercel
```

**Что будет происходить:**
1. Откроется браузер — войди через GitHub (нажми «Continue with GitHub»)
2. Вернись в командную строку
3. На вопросы жми Enter (ответы по умолчанию подходят):
   - «Set up and deploy?» → Enter
   - «Which scope?» → Enter
   - «Link to existing project?» → Enter
   - «Project name?» → Enter
   - «In which directory?» → Enter
   - «Override settings?» → N (нет)

4. Через ~30 секунд сайт будет готов!
5. Vercel покажет ссылку вида: `https://logistics-dashboard-xxx.vercel.app`
6. Скопируй эту ссылку и открой в браузере — твой дашборд в интернете!

---

## Шаг 5: Как обновлять сайт (когда внёс изменения)

Это самое простое. Каждый раз, когда ты меняешь код и хочешь обновить сайт:

1. Открой `Win+R → cmd`
2. Перейди в папку:
```
cd C:\Users\Mike\Desktop\WorkDASHBOARD
```
3. Выполни 3 команды:
```
git add .
git commit -m "Обновление"
vercel --prod
```

Через 30 секунд сайт обновится. Всё!

## Важно: данные на сайте

- Данные хранятся в браузере (localStorage)
- На каждом устройстве (компьютер, телефон) — свои данные
- Чтобы перенести данные на другое устройство: «📥 Бэкап» → сохранить файл → на другом устройстве «📤 Восстановить»
