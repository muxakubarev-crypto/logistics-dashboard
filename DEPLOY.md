# Как разместить дашборд в интернете (Vercel) — с общими данными

Vercel — бесплатный хостинг. Твой дашборд работает по ссылке: **https://logistics-dashboard-rho-sand.vercel.app**

---

## Часть 1: Как обновлять сайт (уже настроено)

Когда внёс изменения в код и хочешь обновить сайт:

1. Открой командную строку: `Win + R` → введи `cmd` → Enter
2. Перейди в папку проекта:
```
cd C:\Users\Mike\Desktop\WorkDASHBOARD
```
3. Выполни 3 команды:
```
git add .
git commit -m "Обновление"
git push
```

Готово! GitHub привязан к Vercel — сайт обновится за 30-60 секунд.

---

## Часть 2: Как поднять сайт с нуля (если делаешь заново)

### Подготовка (один раз)

#### 1. Установи Git
Скачай с https://git-scm.com/download/win → «64-bit Git for Windows Setup» → запусти → везде «Next» → «Install»

#### 2. Зарегистрируйся на GitHub
Зайди на https://github.com → «Sign up» → введи email, придумай пароль и имя

#### 3. Установи Vercel CLI
В командной строке (`Win+R → cmd`):
```
npm install -g vercel
```

### Первый деплой

#### 4. Инициализируй Git в папке проекта
```
cd C:\Users\Mike\Desktop\WorkDASHBOARD
git init
git add .
git commit -m "Первая версия"
```

#### 5. Создай репозиторий на GitHub
- Зайди на https://github.com/new
- Repository name: `logistics-dashboard`
- НЕ ставь галочки (ни README, ни .gitignore)
- Нажми «Create repository»

#### 6. Отправь код на GitHub
```
git remote add origin https://github.com/ТВОЙ_ЛОГИН/logistics-dashboard.git
git branch -M main
git push -u origin main
```

#### 7. Настрой базу данных Vercel KV (для общих данных)

Vercel KV — облачное хранилище. Все гости видят одни и те же данные.

1. Зайди на https://vercel.com/dashboard
2. Найди проект `logistics-dashboard` → нажми на него
3. Вкладка **Storage** (слева в меню) → нажми **Create Database**
4. Выбери **Upstash Redis** (KV) → нажми **Continue**
5. Выбери план **Free** (бесплатно до 256 МБ) → нажми **Create**
6. В появившемся окне нажми **Connect** — Vercel автоматически добавит переменные окружения
7. Готово! KV база создана и подключена к проекту

#### 8. Задеплой на Vercel
```
vercel
```
На все вопросы жми **Enter**. Через 30-60 секунд сайт в интернете!

#### 9. Привяжи GitHub (для авто-деплоя)
- В личном кабинете Vercel: проект → Settings → Git → Connect GitHub repository
- Выбери репозиторий → Connect

## Как это работает (простыми словами)

**Раньше:** данные хранились в браузере. Каждый гость видел пустой дашборд.

**Сейчас:** данные хранятся в облаке Vercel KV. Ты добавляешь месяц → данные уходят в облако. Любой гость, открывший сайт, видит актуальные данные.

- **Админ** (ты) — добавляет/удаляет месяцы через «Управление месяцами»
- **Гость** — видит дашборд с графиками, KPI, подсказками
- **API-ключ DeepSeek** — по-прежнему хранится локально в твоём браузере

## Часто задаваемые вопросы

**Сколько это стоит?**
0 рублей. И Vercel, и Vercel KV бесплатны для личных проектов.

**Данные на сайте?**
Хранятся в облаке Vercel KV. Все гости видят одно и то же.

**Что если API не отвечает?**
Дашборд автоматически использует локальный кэш (localStorage) как запасной вариант.

**Как перенести данные на другой компьютер?**
Не нужно! Данные в облаке — открой сайт с любого устройства, всё там.
