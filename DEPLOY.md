## Как обновить данные (добавить новый месяц)

Данные лежат в файле `src/defaultData.js`. Есть два способа их обновить:

### Способ 1: Через дашборд (как сейчас)

1. На сайте войди как админ (логин `muxakub`, пароль `08250825`)
2. Кнопка «Управление месяцами» → «Добавить месяц»
3. Загрузи Excel или вставь Markdown
4. Нажми «Внести данные»

**Важно:** данные сохраняются в твоём браузере (localStorage). Другие гости увидят ТОЛЬКО предзагруженные данные из файла.

### Способ 2: Обновить файл (чтобы все гости видели)

1. Открой дашборд → войди как админ
2. Нажми «📥 Бэкап» — сохранится файл `kubarev_backup.json`
3. Скажи мне: «Обнови defaultData.js из этого файла»
4. Я обновлю файл → ты сделаешь `git push` → все гости увидят новые данные

## Как поднять сайт с нуля

### Один раз установить

1. **Git**: скачай с https://git-scm.com/download/win → установи (везде Next)
2. **GitHub**: зарегистрируйся на https://github.com
3. **Vercel CLI**: в `Win+R → cmd` введи `npm install -g vercel`

### Первый деплой

```
cd C:\Users\Mike\Desktop\WorkDASHBOARD
git init
git add .
git commit -m "Первая версия"
```
Создай репозиторий на https://github.com/new (название `logistics-dashboard`, без галочек)
```
git remote add origin https://github.com/ТВОЙ_ЛОГИН/logistics-dashboard.git
git branch -M main
git push -u origin main
vercel
```
На все вопросы Enter. Сайт в интернете через 30 секунд.

## FAQ

**Гости видят пустой дашборд?**
Гости видят данные из файла `src/defaultData.js` (11 месяцев). Админ может добавить новые месяцы через дашборд, но они сохраняются локально.

**Как сделать, чтобы все гости видели мои добавленные месяцы?**
Способ 2 выше: сделай бэкап → я обновлю файл → ты сделаешь `git push`.

**Сколько стоит?**
0 рублей. Vercel бесплатный.
