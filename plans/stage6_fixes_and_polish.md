# Этап 6: Исправления + PDF + Редизайн подсказок + AI-контекст

## Диагностика багов

### Баг 1: `test.xlsx` → «Вкладок: 0»
**Корень:** В [`excelLoader.js`](src/excelLoader.js) жёстко закодированы 11 названий вкладок (`июнь 2025`, `июль 2025`...). `test.xlsx` содержит другую вкладку → `availableSheets = []` → пустой Markdown.
**Исправление:** Если ни одна из 11 вкладок не найдена — брать **первую** вкладку из Excel. Или добавить поле «Название вкладки» при загрузке месяца.

### Баг 2: Тёмная тема не выключается
**Корень:** `useEffect` с ночным режимом переопределяет тему каждую минуту. Если пользователь включил тёмную вручную днём — через минуту скрипт включит светлую. И наоборот.
**Исправление:** Добавить флаг `manualOverride`. Если пользователь нажал кнопку темы — больше не применять авто-режим до следующего дня.

### Баг 3: «Мусорные запросы» — термин
Заменить на «Формальные запросы» во всех местах: KPI-карточки, тултипы, умные подсказки, what-if, графики.

// Было:
const availableSheets = MONTH_SHEETS.filter(s => wb.SheetNames.includes(s));

// Стало:
let sheets = MONTH_SHEETS.filter(s => wb.SheetNames.includes(s));
if (sheets.length === 0 && wb.SheetNames.length > 0) {
  sheets = [wb.SheetNames[0]]; // берём первую вкладку
  console.warn('Месячные вкладки не найдены, используется первая:', sheets[0]);
}
