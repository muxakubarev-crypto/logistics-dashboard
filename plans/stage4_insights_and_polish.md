## Задача 3: Мягкие цвета тёмной темы

### Текущая палитра (слишком контрастная)

| Элемент | Сейчас (dark) | Слишком... |
|---------|--------------|-----------|
| Фон | `slate-900` (#0f172a) | тёмный, холодный |
| Карточки | `slate-800` (#1e293b) | контрастный |
| Текст | `slate-100` (#f1f5f9) | яркий |
| Акцент | `blue-600` (#2563eb) | нормально |
| KPI-иконки | `dark:text-blue-400` и др. | нормально |

### Новая палитра (мягче, теплее)

| Элемент | Новый dark-класс | HEX | Почему |
|---------|-----------------|-----|--------|
| Фон страницы | `dark:bg-neutral-900` | #171717 | Теплее slate, меньше синевы |
| Карточки | `dark:bg-neutral-800` | #262626 | Мягкий контраст с фоном |
| Вторичный фон (textarea, select, инпут чата) | `dark:bg-neutral-850` (кастом: #1f1f1f) | #1f1f1f | Промежуточный оттенок |
| Границы | `dark:border-neutral-700` | #404040 | Меньше контраста |
| Текст основной | `dark:text-neutral-100` | #f5f5f5 | Чуть теплее белого |
| Текст вторичный | `dark:text-neutral-400` | #a3a3a3 | Мягче |
| KPI-фон иконок | `dark:bg-neutral-700` | #404040 | Вместо slate-700 |
| Ховер на кнопках | `dark:hover:bg-neutral-700` | #404040 | Мягче |
| Скелетоны | `dark:bg-neutral-700` | #404040 | Мягче |
| Чат — сообщения юзера | `bg-blue-600` (без изменений) | #2563eb | Акцент |
| Чат — сообщения AI | `dark:bg-neutral-700` | #404040 | Мягче |

### Сравнение

```
Было:  slate-900 → slate-800 → slate-700 (холодные, синеватые)
Стало: neutral-900 → neutral-800 → neutral-700 (тёплые, серые)
```

### Дополнительно: кастомный оттенок `neutral-850`

Tailwind 4 не имеет `neutral-850` из коробки. Добавим в `index.css`:
```css
@layer base {
  .dark .bg-neutral-850 {
    background-color: #1f1f1f;
  }
}
```
